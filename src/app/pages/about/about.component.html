Sobre o Projeto

Este projeto foi desenvolvido como parte do aprendizado do módulo de Angular na DIO (Digital Innovation One).

O objetivo principal foi aplicar os conhecimentos em Angular, como componentes, serviços, rotas e consumo de APIs REST, para construir um blog de notícias.

As notícias são obtidas através da API pública thenewsapi.com, que oferece uma variedade de artigos jornalísticos em tempo real.

Este blog simula um portal de notícias moderno, com funcionalidades como listagem, detalhamento e sugestões de notícias relacionadas.
